# 🛒 Kasir Supermarket CLI

Project akhir Python Fundamental Programming berbasis CLI (Command Line Interface).

---

# 📌 Deskripsi Project

Kasir Supermarket CLI adalah aplikasi kasir sederhana berbasis terminal menggunakan bahasa Python.  
Program ini dibuat untuk membantu proses transaksi supermarket seperti melihat daftar barang, menambahkan pesanan, menggunakan diskon member, mencetak struk, dan menghitung kembalian.

Project ini dibuat untuk memenuhi tugas Project Akhir KKA Python Fundamental Programming.

---

# ✨ Fitur Program

✅ Menampilkan daftar barang berdasarkan kategori  
✅ Menambahkan pesanan barang  
✅ Mengurangi stok otomatis  
✅ Diskon member 10%  
✅ Menghitung pajak 5%  
✅ Cetak struk belanja  
✅ Menghitung kembalian pembayaran  
✅ Tampilan menu CLI interaktif  
✅ Program berjalan terus sampai user memilih keluar
